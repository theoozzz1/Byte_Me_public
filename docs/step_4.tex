\section{Step 4 – Run Automated Tests}

Markers can run the automated tests for the backend using Maven.

\subsection{Run Backend Tests}

From the project root:

\begin{lstlisting}
cd backend
mvn test
\end{lstlisting}

This will execute all unit and integration tests using the Maven Surefire plugin.

Expected output will end with something similar to:

\begin{lstlisting}
BUILD SUCCESS
\end{lstlisting}

If any tests fail, Maven will display the failing test names and stack traces.

\subsection{Run a Specific Test Class}

Markers can also run a single test class:

\begin{lstlisting}
mvn -Dtest=ClassNameTest test
\end{lstlisting}

Example:

\begin{lstlisting}
mvn -Dtest=IssueControllerTest test
\end{lstlisting}
